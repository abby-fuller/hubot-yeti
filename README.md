# hubot-yeti

all the yetis for hubot

See [`src/yeti.coffee`](src/yeti.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-yeti --save`

Then add **hubot-yeti** to your `external-scripts.json`:

```json
["hubot-yeti"]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
